The system is created using visual studio asp.net c# programming language.
you will require to have Visual Studio (preferably VS 2013 ) installed.

HOW TO RUN:
Install VS into your system.
Open the Microsoft Visual Studio Solution (.sln) "bluline" in the folder (Blueline).
when open press F5 to run the solution.

HOW TO LOGIN AS ADMIN
use the following credentials:
	Username: admin
	Password: admin

HOW TO LOGIN AS AGENT/USERS
you can create new users and agents or use the credentials you find under the userinfo database in the bluline.mdf
